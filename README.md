# HTML Project Skeleton
***Lightweight starter for html projects.***
  - Bootstrap
  - Sass / JS merge
  - Live reload with Browser Sync

### Installation
```sh
$ git clone https://github.com/mrmoonlv/html_skeleton.git
$ cd html_skeleton
$ npm install
$ bower install
$ gulp
```
